# vuetify-tags-component

# Скрипты

## Установка зависимостей

```
yarn install
```

### Dev-режим, старт

```
yarn serve
```

### Билд

```
yarn build
```

### Линтер

```
yarn lint
```

### Кастомная конфигурация

См. [Configuration Reference](https://cli.vuejs.org/config/).
